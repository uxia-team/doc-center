The **Usage** tab gives you visibility into organization-level activity and credit consumption so you can keep an eye on capacity and adoption.

## What You Can See

- Credit usage for the current billing period.
- Remaining credits, or confirmation that your plan has unlimited credits.
- Total tests launched across the organization.
- A per-user usage breakdown.
- A per-workspace usage breakdown.

## Activity Breakdown

The usage tables show activity across key product areas, including:

- Total launched
- AI User Test usage
- Accessibility usage
- Audiences usage

For more on how credits work and what's included in each plan, see [Plans & Credits](/account/plans-and-credits).
