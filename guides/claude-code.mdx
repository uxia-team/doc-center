## Setup

1. Run the following command in your terminal to register the Uxia MCP server:
   ```bash
   claude mcp add uxia --scope user --transport http https://platform.uxia.app/api/mcp/mcp
   ```
   bash

Start a new Claude Code session in your terminal. In the list of MCP servers, select **Uxia**.

```bash
claude /mcp
```

1. Select **Authenticate**.
2. A browser window will open automatically. Sign in with your Uxia account to authorize access.

## Usage

Once connected, you can ask Claude perform actions in Uxia directly:

```plaintext
Create a new test thats tests my new app www.google.com 
```
