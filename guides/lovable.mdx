## Setup

1. In Lovable, go to **Settings \> Connectors \> Personal connectors**.
2. Click **New MCP server**.
3. Enter **Uxia** as the name.
4. Paste the server URL:
   ```plaintext
   https://platform.uxia.app/api/mcp/mcp
   ```
5. On first use, a browser window will open automatically. Sign in with your Uxia account to authorize access.

## Usage

Once connected, you can ask Lovable to talk to Uxia directly:

```plaintext
Create a new test thats tests my new app www.google.com 
```
