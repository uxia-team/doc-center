## Setup

The Codex desktop app shares credentials with ChatGPT. Set up Uxia in [ChatGPT](/mcp/clients/chatgpt) first, and the Codex app will automatically have access.

## Usage

Once connected, you can ask Codex App to talk to  Uxia directly:

```plaintext
Create a new test thats tests my new app www.google.com 
```
